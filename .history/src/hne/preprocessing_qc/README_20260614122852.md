# Preprocessing QC Module

Tracks quality metrics and generates diagnostic plots for preprocessing pipeline.

## Module Structure
```text
preprocessing_qc/
├── tracker.py # QC record keeping & summaries
└── plots.py # Visualization functions
```

## QCTracker (`tracker.py`)

Collects and saves QC records throughout preprocessing.

### Methods

| Method | Purpose |
|--------|---------|
| `add_record()` | Add QC event (PASS/WARNING/ERROR) |
| `save_summary()` | Generate exclusion list for cohort |
| `save_metadata()` | Export all QC metadata to CSV |
| `save_cohort_spot_qc_plots()` | Aggregate plots across patients |

### Record Fields

- `patient_id` - Patient identifier
- `timestamp` - ISO format datetime
- `stage` - Pipeline stage (tumor_fraction, tiling, etc.)
- `status` - PASS, WARNING, ERROR, SKIPPED
- `message` - Human-readable description
- `metadata` - Optional numeric metrics

### Outputs
qc_reports/preprocessing_qc/{mode}/
├── qc_summary.csv # Per-patient exclusion status
├── metadata.csv # All QC metrics
├── signature_variation_{id}.png
├── signature_distribution_{id}.png
├── signature_sparsity_{id}.png
├── signature_consistency_{id}.png
└── signature_correlation_{id}.png


## Plots Module (`plots.py`)

### Available Plots

| Function | Purpose | Key Metrics |
|----------|---------|-------------|
| `signature_variation()` | Standard deviation across spots | Low variation → poor MIL features |
| `signature_distribution()` | Density plots | Identifies skewed signatures |
| `signature_sparsity()` | Fraction of near-zero scores | High sparsity → sparse features |
| `signature_consistency()` | Gene-gene correlation within signature | Low consistency → noisy signature |
| `signature_correlation()` | Cross-signature correlation | High correlation → redundant features |


## Usage Example

```python

from hne.preprocessing.pipeline import preprocess_patient

qc = QCTracker(mode='single_patient')

metadata, tiles_sig, spots_df, sig_cols = preprocess_patient(
        "CH_L_282", 
        mode='single_patient',
        tile_size=100,        # in pixels, ≈ 1 mm in hires image
        k=2, 
        tumor_threshold=0.3,  # tile at least has 30% tumor purity
        min_spots=40,         # tile at least has 40 spots
        qc_tracker=qc,
        verbose=True,
        run_qc_plots=True
    )

qc.save_metadata(metadata)
summary = qc.save_summary()
