# Preprocessing Module

Core preprocessing pipeline for aligning spatial transcriptomics with H&E tiles.

## Module Structure

```text
preprocessing/
├── pipeline.py # Main orchestration
├── tumor_purity.py # Compute Bayesian purity per tile, filter tumor tiles 
├── tiling.py # Crop H&E and tile extraction
├── spot_signatures.py # Compute signature scores for spots
└── aggregation.py # Aggregate spot→tile signatures, z-score & binary
```


## Pipeline Flow
Raw Data → tumor_purity → tiling → spot_signatures → aggregation → Outputs


### 1. Tumor Purity (`tumor_purity.py`)

**Functions:**
- `attach_tumor_fraction()` - Add tumor fraction to each spot from deconvolution results
- `add_tile_coordinates()` - Assign each spot to a tile based on spatial coordinates
- `compute_tile_purity()` - Bayesian estimation of tile-level tumor purity
- `filter_tumor_tiles()` - Select tiles meeting purity & spot count thresholds

**Parameters:**
- `k=2` - Bayesian prior strength (higher = more regularization)
- `tumor_threshold=0.3` - Minimum 30% tumor cells in tile
- `min_spots=40` - Minimum spots for statistical reliability

### 2. Tiling (`tiling.py`)

**Function:** `crop_and_save_tiles()`

Extracts H&E image patches at calculated tile coordinates.
- Input: Tumor tile DataFrame + H&E image
- Output: PNG tiles saved to `results/preprocessing/tiles/{patient_id}/`
- Resolution: 600 DPI for MIL compatibility

### 3. Spot Signatures (`spot_signatures.py`)

**Function:** `compute_signatures()`

Calculates five pathway activation scores per Visium spot using ScanPy.
- Uses log-normalized expression counts
- Reports missing genes as warnings
- Returns z-scored signature matrix

### 4. Aggregation (`aggregation.py`)

**Functions:**
- `aggregate_signatures()` - Mean aggregation of spot signatures per tile
- `zscore_and_binary()` - Normalize and binarize tile signatures

**Output:** Tile × signature matrix with:
- Raw scores
- Z-scores (per signature)
- Binary calls (Z ≥ 1.0)
